#### 课程1.19 清洗（简介）
编程数据清洗过程：

1. 定义

	> 指以书面形式定义数据清洗计划，其中我们需将评估转变为定义的清洗任务。这个计划也可作为一个指导清单，所以其他人（或我们自己将来）也可以回顾和重现自己的工作。
2. 代码

	>编码指将这些定义转换为代码并执行该代码。
3. 测试

	>测试指练习我们的数据集，通常使用代码，以确保有效完成我们的清洗工作。
